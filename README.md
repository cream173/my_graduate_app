## 1. サービス概要（※最後に書いてOK）
本アプリは、日々の体調や気分を記録し、体の状態に合わせた養生法や食材を提案するセルフケアサポートサービスです。
忙しく自分のことを後回しにしがちな人でも、体調の変化に気づきやすくすることを目指します。
体調の可視化と提案を通して、自分に合った健康習慣を見つける手助けをします。

## 2. このアイデアはどこから生まれたか

### 2-1. きっかけとなった体験・感情
このアプリを作ろうと思ったきっかけは、家族の知人が突然病気と診断された出来事です。
後から話を聞くと、思い返せば診断される数ヶ月前から「よくお腹が痛いと言っていた」「顔色が悪いようにみえた」など、体調の変化が見られていたそうです。
その話を聞いてからもっと早い段階で体調の変化を自覚し病院を受診できていたら結果は違っていたのではないか、と考えるようになり、体調を主観的に記録できるアプリを開発したいと思うようになりました。

### 2-2. なぜそれが「気になった」のか
家族の知人の出来事をきっかけに、自分自身や周囲の生活を振り返ってみると、

- 頭が痛い
- 疲れがたまっている気がする
- なんとなく体調が悪い

などと感じていても、休めば治るだろうと考えそのまま日常を過ごしてしまうことが多いことに気づきました。
記録をすることで、「ここ数日なんとなく頭が痛いと思っていたら実は1ヶ月以上続いていた」といった体の感覚とのミスマッチを可視化でき体調の変化の気づきをサポートできるのではないかと考えました。
また、体調が優れない状態が続くと、気分も落ち込みやすくなり、仕事や日常生活への集中力やパフォーマンスにも影響が出ることがあり、生活全体の質も下がってしまうこともあるかと思います。
私自身漢方や薬膳に興味があり、体調の小さな変化に合わせて生活習慣や食事を整える「養生」という考え方は毎日を快適に過ごす上で大切であると感じています。
そのため、日々の体調を記録し体調に合わせた養生法を取り入れることが習慣化されればより体調の変化に気づきやすくなり健康への意識を自然と高めることができるのではないかと考えました。

## 3. 課題の整理（表面的になっていないか）

### 3-1. 表に見えている困りごと
健康は大切であると感じているが忙しい日常だと毎日意識することは難しい

### 3-2. 本当に解決したい課題は何か
- なぜ？
    - 体調の小さな不調サインを見逃してしまうおそれがある
- なぜ？
    - 不調が悪化する恐れがある
- なぜ？
    - 日常生活に支障が出たり、回復に時間がかかる

→ 本当に解決したい課題：体調の小さなサインに気づけない

## 4. 想定ユーザーについて

### 4-1. 想定しているユーザー
年齢設定：

- 年齢：主に30-40代の働き盛りで忙しい年代の方
- 理由：仕事や家事などで忙しく、自分の体調管理を後回しにしてしまうことが多い一方で将来の健康についても意識し始める年代であるため

生活・状況設定：

- 生活・状況：忙しい日常の中で食事や生活習慣が乱れやすく、体調が優れないと感じてもそのまま過ごしてしまうことが多い
- 理由：体調を意識する機会が少なく、小さな不調に気づきにくいことが課題となるため、体調の記録や可視化によって自分の状態を客観的に把握できるようにする

環境設定：

- 環境：仕事や子育て、家事に追われ忙しい毎日を送っている
- 理由：毎日の記録は簡単にできるようにする、労いのメッセージの表示

### 4-2. 自分とユーザーの距離
自分自身

理由: 自分自身も体調の小さな変化を見過ごしてしまうことがあり、セルフケアを意識して生活することの大切さを感じたため
